1. Introduction au projet

Objectif : Créer une application qui facilite le déplacement des personnes âgées pour leur trajet du quotidient. Contenu : Notre projet est une application de voiture autonome pour les personnes âgées. Simplifier le déplacement des personnes âgées pour leur course habituelle.
